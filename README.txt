== Description ==

Vergelijkdirect.com vergelijkt verzekeringen, energie en telecomaanbieders. 
Met deze simpele plugin ben je in staat om vergelijkingsmodules op je website te 
plaatsen om je bezoekers een onafhankelijke vergelijking aan te kunnen bieden!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `vdpetform.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit your plugin inside the vergelijkdirect.com admin menu
4. Volg de instructies op in de plugin voor het plaatsen van een vergelijker
